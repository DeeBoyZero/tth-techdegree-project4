# Team Treehouse FSJS unit 4 project
### by Mathieu Desilets

## A simple phrase hunter game.
Player must try to find the hidden phrase by guessing letters using his keyboard or his mouse with the visual keyboard.


### Techno used
- HTML
- CSS
- Javascript
